Samba

Setup

In /etc/samba/smb.conf:

  1. Change Workgroup to SCHAU.
  2. Add this section: 
    [data]
        path = /data
        browsable = yes
        guest ok = yes
        read only = yes
        write list = bs
        create mask = 0600
        directory mask = 0700
	valid users = bs cfs


Adding user

As root:

cd /var/lib/samba/private
smbpassword -a {username}


Connect from client

Map drive, use schau\bs as username.
