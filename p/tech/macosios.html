Cannot execute file

If permissions ends in '@' - check which permissions:

ls -@l file

if ACL attributes include com.apple.quarantine then:

sudo xattr -d com.apple.quarantine file

... to remove it. 'File' can now be run.


"Circle to Search" on iOS

No, this will not give you Circle to Search as found on the latest Androids from Google and Samsung. But it's kinda close. I am not the inventor - but I found the recipe on CNET (https://www.cnet.com/tech/services-and-software/googles-circle-to-search-is-now-on-the-iphone-well-kinda/. Thank you, CNET!

Requirements

  • Newer iOS
  • The Shortcuts app installed
  • The Google app installed (we're going to use the Lens part of it)


Howto

  1. Open the Shortcuts app.
  2. Hit the + button in the top-right.
  3. Tap Add Action (Tilføj handling).
  4. Search for and tap Take Screenshot (Tag skærmbillede).
  5. Hit the search bar at the bottom.
  6. Search for and tap Search Image with Lens (Søg efter billedet med Lens).
  7. Hit Done.

If you have an iPhone 15 or newer you can bind this to your Action button in Settings.

I only have a iPhone 13 Mini (awesome phone!) so I am using the back tap functionality:

  1. In Settings, go to Accessibility → Touch → Back Tap.
  2. Select either Double tap or Triple tap.
  3. Select the Search Image with Lens shortcut (Søg efter billedet med Lens).
